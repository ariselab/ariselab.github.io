ARiSE Columbia University




ARiSE Lab is an Academic Research Lab led by Prof. Baishakhi Ray at Columbia University, Department of Computer Science.

ARiSE (Advanced Research in Software Engineering) lab aims to develop tools that automate software development and maintenance for real-world large scale software. In particular, we are interested in testing and debugging large-scale real-world systems—ranging from traditional logic-based systems to more recent AI-driven systems like self-driving cars, malware detection tools, etc. We also build novel program analysis techniques to analyze existing system properties and apply advanced neural network models to learn from those properties. Such models are then used to automate development, testing, debugging, and fixing of real-world large-scale software.
